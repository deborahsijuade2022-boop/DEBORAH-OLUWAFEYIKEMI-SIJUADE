# ✨ Debby — My Little Corner of the Web

A single-page personal website built with **React + Vite**, featuring a playful pink aesthetic and components organized across multiple folders.

## 🗂️ Project Structure

```
src/
├── components/
│   ├── layout/          # Navbar, Footer
│   ├── sections/        # Hero, About, Hobbies, Contact
│   └── ui/              # Button, Card (reusable components)
├── App.jsx              # Root component
├── App.css              # Global CSS variables & resets
├── index.css            # Font imports & base styles
└── main.jsx             # React entry point
```

## 🚀 Getting Started

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```

## 🎨 Features

- **Navbar** — Fixed navigation with smooth underline hover effects
- **Hero** — Animated gradient blobs, avatar, and call-to-action buttons
- **About** — Info cards with staggered fade-in animations
- **Hobbies** — Interactive chips with hover pop effects
- **Contact** — Styled form with focus states
- **Footer** — Floating emoji footer

- [React](https://react.dev/)
